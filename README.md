# Goose UML for VS Code

Generate UML class and sequence diagrams for TypeScript, JavaScript, Java, and Python code directly in VS Code.

## Features

- **Class Diagrams**: Visualize class structures, relationships, and hierarchies
- **Sequence Diagrams**: Understand method call flows and interactions
- **Multi-Language Support**: TypeScript, JavaScript, Java, and Python
- **Cross-File Analysis**: Analyze dependencies across multiple files
- **Interactive Controls**: Adjust depth and analysis mode in real-time

## Usage

1. Open a supported file (TypeScript, JavaScript, Java, or Python)
2. Press `Ctrl+Shift+U` (or `Cmd+Shift+U` on Mac) to open the UML panel
3. Select diagram type (Class or Sequence)
4. Adjust analysis depth and mode as needed
5. View, zoom, and export your diagrams

## Commands

- `Goose UML: Open UML Panel` - Open the interactive UML diagram panel
- `Goose UML: Generate Class Diagram` - Generate a class diagram
- `Goose UML: Generate Sequence Diagram` - Generate a sequence diagram
